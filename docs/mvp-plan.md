# MVP Plan

## Phase 1

Read a local project and generate:

- `repo-summary.json`
- `project-insight.json`
- `script.md`

## Phase 2

Generate a constrained `storyboard.json` with 6 to 8 scenes.

## Phase 3

Render a silent 9:16 video from a fixed Remotion template.

## Phase 4

Generate Chinese TTS and adjust scene durations from measured audio length.

## Phase 5

Add a verifier step to reduce unsupported claims and exaggerated wording.

## Phase 6

Support public GitHub URLs through clone-to-cache and reuse the local project pipeline.
